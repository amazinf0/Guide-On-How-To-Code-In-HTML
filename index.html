Learning To Code

Basics


To make a pressable link through text, use the A tag and inside it use the href tag and put the link you want the text to go to in the quatation marks. Use "Targets="_blank" to make the link take you to the page through the same tab rather than opening a new one.  Example

To make text go back and forth, use a Marquee tag and you can change the directon and speed by adding a behaviour and direction tag inside the marquee.

Style is a tag that's placed either in a tag or directly under meta data if you want it to affect everything, it is used for flexbox, backgrounds, fonts, and DIVs.

To make the splashpage title, use a title tag under the keyboard language and enter your preferred title.

It is also advised to make your code look nice and easy to read by pressing space and extending it out so it would almost be like a sandwhich.

The ul tag makes a list which you can add bullet points to by using the li tag, the shape can be customized from the ul tag.

Using the & symbol and adding some text can allow you to do things like add a space in the text or make a copyright symbol © .

To make a background from the style, you need to put the image link rather than the downloaded image's pathway.

The best way to learn how to code is to just start doing it yourself and like me, you're gonna be able to code on your own but, not be able to understand what you're looking at but it works so it's fine.

To make the text change in size according to the size of the webpage, use the style tag and use fontsize to allow that to happen.

To center the text, in the style which is in the semantic tag, use text-align: center to center the text.
