Meta Data


Summary of Meta Data:








The meta data is a section where you include things like keywords so that someone can find your website if they searched it up.

It is also where you can decide what type of keyboard style is used on it. EX. QWERTY

The meta data also informs the viewer who the auther is and the purpose of the website.

To make a note in the code that can only be seen in the code, use a < !-- --> tag.
