Summary of Semantic Tags


The body tag is what all of the following tags should be within.

The header tag is the header of the page. An example of this is the title of this webpage.

The navigation is where all the buttons tend to be placed.

The aside tag tends to be something to the side with less importance than other things.

The main tag is basically the body of the website and there should be one per website.

The section tag is where things that are similar to eachother are placed so as to be grouped together.

The article tag tends to be something like a blog that could stand on it's own.

The footer usually has a blockquote in it and also is where contact information and media things tend to go.
