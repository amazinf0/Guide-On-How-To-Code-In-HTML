Flexbox


Summary of Flexbox:








Flexbox is a container where you put text or images which can change in size and positioned to your liking.

A flexbox can have a border, background color, and unique size.

To change all of those things, go to the style tag and choose a box sizing and a font size and then put a dot and then the tag. ex. main.

You can also choose to make it so that you need to specifically put a class in a semantic tag so it chooses main by doing .main , if you don't do that, it will affect any semantic tag that is using the tag main.

From there, you can choose the empty space and border size/color and background color, you can also make the flexbox be on a certain side by writing float:(direction).

To get an example of all of this, open the WYSIWYG and look at the style tag.
