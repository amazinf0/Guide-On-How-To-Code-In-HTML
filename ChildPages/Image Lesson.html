Images


Summary of Images:








An img tag is a self closing tag which allows you to place an image into your webpage.

To put an image into the img tag you need to download it and paste the directory into the src and alt section so the image is displayed.

To name the image, put a title tag inside the img tag and that would be your name.

To change the size of the image, use the style tag inside the img tag and change the width and height to your liking.

To force an image to be somewhere, use flexbox.

To make a background, add a background tag in the body and point it to the image and viola!
